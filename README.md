# Minesweeper WPF

Acesta este un joc clasic Minesweeper dezvoltat folosind .NET WPF, realizat în perioada liceului. Jocul reproduce mecanicile originale Minesweeper, oferind o interfață grafică interactivă și logică completă pentru generarea și gestionarea câmpului de mine.

Proiectul a fost realizat pentru a învăța dezvoltarea aplicațiilor desktop folosind Windows Presentation Foundation și concepte de programare orientată pe obiect.

---

## Tehnologii utilizate
- **Framework:** .NET  
- **UI Framework:** WPF (Windows Presentation Foundation)  
- **Limbaj:** C#  
- **IDE:** Visual Studio 

---

## Funcționalități principale
- Generarea aleatorie a minelor pe grid  
- Detectarea click-urilor stânga și dreapta  
- Sistem de flag pentru marcarea minelor  
- Dezvăluirea automată a celulelor goale (flood fill)  
- Detectarea condițiilor de win și game over  
- Interfață grafică interactivă realizată în WPF  

---

## Ce am învățat
- Dezvoltarea aplicațiilor desktop folosind WPF și C#  
- Lucrul cu controale UI, layout-uri și evenimente  
- Implementarea logicii unui joc bazat pe grid  
- Gestionarea evenimentelor de mouse  
- Organizarea codului folosind programare orientată pe obiect  
- Separarea logicii jocului de interfața grafică
